### Installation:
```
cd [project_folder]
git clone https://github.com/thohoh/ng-leaflet-test
cd ng-leaflet-test
bower i
npm i
```

### Run app:
```
grunt serve
```
Now navigate your browser to http://localhost:3000/src/index.html

```
grunt serve // serve static files
